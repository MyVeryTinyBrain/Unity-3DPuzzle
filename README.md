## Build  
https://github.com/MyVeryTinyBrain/Unity-3DPuzzle/releases/download/build/Puzzle.zip  
실행파일: `Puzzle.exe`

## How to play  
이동: `W` `A` `S` `D`  
아이템 줍기, 오브젝트 선택: `Left mouse click`  
오브젝트 선택 취소: `Tab` OR `Escape`  
들고있는 아이템 살펴보기, 취소: `E`
